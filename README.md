# BaseBall_Self_Management

PHP自作

## 概要

中学〜高校生をメインとした学生野球の管理システムとなっており

選手自身が体力の成長や体調を管理でき、周りの選手にも共有ができるものとなっております。

## 使い方

ログイン画面よりログイン・選手登録・編集・削除等ができます。
